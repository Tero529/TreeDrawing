Relative Positioning of Nodes.

Performs a post order traversal recursively along the left and right contours of the input tree

Parameters
tThe node whose left and right subtree's contours are to be followed
levelthe height of the node passed
rmostpointer which is to be set with the right most node in the subtree starting from given node
lmostpointer which is to be set with the leftmost node in the subtree starting from given node

Threading is done if the left and right subtrees are of differing heights
