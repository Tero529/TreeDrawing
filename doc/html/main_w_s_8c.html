Detailed Description

Main Driver for Wethrerell Shannon Drawing.

This file begins execution by creating a Binary Search Tree, and feeding it through the Wetherell Shannon Tree Placement Algorith before using the generated x and y values to draw a tree to screen using the OpenGL API for drawing and GLUT for windowing

Startpoint of Application.

First runs createTree() to create tree to be drawn

It then runs Tree Drawing Algorithm in "WetherellShannon.h" using the root of the tree generated, the height of the tree and the type of second pass required

WIDTH and HEIGHT are given given pixel values so as to yield a window that just fits the given tree

Uses GLUT to initiliaze window with the created WIDTH and HEIGHT and places window at top left of screen

Then Initialise Drawing settings wiith Init()

Function to render objects to screen.
