\subsection{Detailed Description}
Algorithms for Line, Circle Drawinng and Circle Fill. 

Header Guard to Prevent Redeclaration

This file contains functions to translate coordinates and to drawn primitives on the screen using
\begin{DoxyEnumerate}
\item Open\+GL A\+PI for pixel drawing
\item Generalised Bresenham\textquotesingle{}s Midpoint Algorithm for Line Drawing
\item Midpoint Circle Algorithm
\item 4 Neighbour Flood Fill, for filling circles 
\end{DoxyEnumerate}
