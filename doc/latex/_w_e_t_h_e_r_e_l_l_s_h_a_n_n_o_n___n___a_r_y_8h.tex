\subsection{Detailed Description}
Extension of Wetherell Shannon to Accomodate N-\/ary Trees. 

Through this file we have extended the algorithm put forth in the paper by Wetherell Shannon for positioning nodes of a binary tree to accomodate an arbtriry number of children.
