\subsection{Detailed Description}
Extension to Accomodate N-\/ary Trees. 

Through this code we have extended the algorithm by taking inspiration from the papers by Wetherell Shannon and tilford Reinfold for positioning nodes of a binary tree to accomodate an arbitrary number of children. As such it draws an n-\/ary tree using many of the same concepts as the aforementioned algorithms. \begin{DoxySeeAlso}{See also}

Recursive Preorder Drawing Walk. 

Uses coordinates generated to Draw the given tree ~\newline
 Walks along the tree in preorder and drawing each edge from node being traversed to its parent, before going to it\textquotesingle{}s sibling. Once all sibling\textquotesingle{}s edges are drawn, nodes are drawn in the reverse order, before moving on the parent. Does not draw the root node, which is drawn from \hyperlink{main_8c_a6765cdcebe5c8445969d5f76345df457}{render(void)};
