\subsection{Detailed Description}
Main Driver for Wethrerell Shannon Drawing. 

This file begins execution by creating a Binary Search Tree, and feeding it through the Wetherell Shannon Tree Placement Algorith before using the generated x and y values to draw a tree to screen using the Open\+GL A\+PI for drawing and G\+L\+UT for windowing
