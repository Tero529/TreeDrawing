\subsection{Detailed Description}
Wetherell Shannon Tree Drawing Algorithm. 

This file contains the functions required to implement the Wetherell Shannon tree Drawing Algorithm. There is support for both the normal version through the N\+O\+R\+M\+AL Algorithm\+Type as well as the narrower version which provides minimum width without ensuring that parents are centered over their children through the M\+O\+D\+I\+F\+I\+ED Algorithm\+Type enumeration

\item Has Left son, place to its right
\item Has Right Sons, place to its left
