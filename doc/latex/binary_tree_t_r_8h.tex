\subsection{Detailed Description}
Tree Functions for Tilford Reingold. 

Specifies functions to create a binary search tree from user input data or hard coded data (See source code for \hyperlink{binary_tree_t_r_8h_a4aebe922a43b1f8500db00465ee139be}{create\+Tree()};) and sets all tree related fields to be used by the Tilford Reingold Tree Drawing Algorithm, Also provides functions for translating integral Wetherell Shannon outputs to window coordinates for drawing.
