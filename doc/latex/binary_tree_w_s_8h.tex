\subsection{Detailed Description}
Tree Functions for Wetherell Shannon. 

Specifies functions to create a binary search tree from user input data or hard coded data (See source code for \hyperlink{binary_tree_w_s_8h_a26673d37c93421a354fb6bb44fab83eb}{create\+Tree()};) and sets all tree related fields to be used by the Wetherell Shannon Tree Drawing Algorithm, Also provides functions for translating integral Wetherell Shannon outputs to window coordinates for drawing.
