Detailed Description

Tree Functions for Wetherell Shannon.

Specifies functions to create a binary search tree from user input data or hard coded data (See source code for createTree();) and sets all tree related fields to be used by the Wetherell Shannon Tree Drawing Algorithm, Also provides functions for translating integral Wetherell Shannon outputs to window coordinates for drawing.

Translates Wetherell Shannon Integral x-value into Window coordinate.

Each integral coordinate is scaled by 100 to be visible appropriately in the screen (screen is typically 800x600 or greater), and is off set on the left by 50 pixels (as node radius drawn is 25 pixels)
